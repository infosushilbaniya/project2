#  Responsive Navigation Bar (HTML + CSS)

This project demonstrates how to create a **simple and clean navigation bar** using **HTML** and **CSS only** — no JavaScript required.  
It’s a beginner-friendly project that focuses on using **flexbox layout**, **hover effects**, and **semantic HTML** to build a professional navbar.





##  Features

-  Built entirely with **HTML5** and **CSS3**
-  Custom hover effect with background color transition
-  Responsive layout using **Flexbox**
- Highlighted brand icon/title on the left
- Lightweight and easy to understand


##  Technologies Used

- **HTML5**
- **CSS3 (Flexbox)**
